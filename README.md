# Engineer Student Struggle Poem Generator


Welcome to the Engineer Student Struggle Poem Generator project! This project is a simple Node.js application that utilizes the Puppeteer library to automate browser tasks and generate AI-powered poems. The poems generated by this application revolve around the theme of struggles faced by engineering students.

## Introduction

As an engineer student, we often find ourselves facing various challenges and struggles in our academic journey. This project aims to capture those experiences through AI-generated poems. The application uses the Puppeteer library to automate interactions with a web-based AI generator, which then produces poems based on the theme of engineering student struggles.

## Installation

1. Clone this repository to your local machine using:

   ```
   git clone https://github.com/your-username/engineer-poem-generator.git
   ```

2. Navigate to the project directory:

   ```
   cd engineer-poem-generator
   ```

3. Install the required Node.js dependencies:

   ```
   npm install
   ```

## Usage

1. Make sure you have Node.js installed on your system.

2. Open the `index.js` file in a text editor of your choice to see the code and make any necessary adjustments.

3. Run the application using the following command:

   ```
   node index.js
   ```

4. The application will automate the process of generating a poem related to the struggles of engineering students using an AI-powered generator.

## Configuration

You can modify certain parameters within the `index.js` file to customize the poem generation process:

- **`aiGeneratorURL`**: This variable holds the URL of the AI-powered poem generator. If the URL changes or if you want to use a different generator, update this variable accordingly.

- **`poemTopic`**: This variable specifies the topic of the poem. By default, it is set to "engineer student struggle," but you can change it to explore different themes.
